## Scripts description

**Data-preparation.R** - Harmonizing trait measurment and metadata columns across collection years.

**Normality-check.R** - Shapiro test on trait measurments.

**trait points.R** - Computing the total data points.

**Location-map.R** - Germany map highlighting Spiekeroog and Brachwitz.

**FAMD-PCA.R** - Clustering analyses using factor analysis of mixed data and principal component analysis.

**Group-comparisons.R** - Two groups pairwise comparisons across collection years, seasons and locations.

**MAinshoot.R** - Two groups pairwise comparison between plants with and without the mainshoot.

**Correlations - Latitude removed.R** - Correlation heatmaps.

**Mixed models.R** - mixed linear models of trait:trait relationship for spring collection combining spiekeroog and brachwitz with colection year and location as random effect variables.
