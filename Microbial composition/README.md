# Scripts description
**script_kaiju.sh**: Kaiju was used to classify the reads that did not mapped against the _A. thaliana_ genome. Kaiju version 1.10.1 was used.

**transformation_table.R**: The _tsv_ table generated from Kaiju was transformed to be evaluated with Phyloseq.

**Phyloseq_Github.R**: Phylosq version 1.48.0 was used to generate the microbial community composition plot (Figure 3b).
